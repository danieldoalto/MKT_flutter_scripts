### 🚀 Quick Start

#### **Windows:**
1. **Extract** and run `myapp.exe`.
2. Configure your routers in the **Settings** tab via the new Editor.

#### **Android:**
1. **Install** `app-release.apk` on your device.
2. Grant network and storage permissions.
3. Add your routers in the Settings panel.
